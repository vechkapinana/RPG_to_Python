Документация проекта: RPG/LLM конвертер в Python

Основная цель проекта
Проект предназначен для автоматической трансляции логики RPG-кода в исполняемый Python-код с возможностью использования LLM для генерации и анализа кода.
Он включает:
- **Парсинг RPG-кода**
- **Конвертация в семантическую структуру** 
- **Добавление анкорных идентификаторов**
- **Генерация Python-кода** 
- **Анализ и генерация через LLM**
